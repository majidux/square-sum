# Square sum


### [npm link](https://www.npmjs.com/package/square-sum)

#### How to install
###### With npm:
```javascript
npm install --save square-sum
```

###### With yarn:
```javascript
yarn add square-sum
```

#### How to use:
```javascript
import { squareSum } from 'square-sum';
const fib = squareSum([10, 3, 4])
```
